# Quick Start

Get from zero to a running agent in under a minute.

## 1. Create a Project Directory

```bash
mkdir my-squadron && cd my-squadron
```

## 2. Run the Wizard

```bash
squadron quickstart
```

The wizard walks you through:

- Picking a provider (Anthropic, OpenAI, or Gemini)
- Entering your API key (stored in an encrypted vault)
- Generating a starter mission that fetches the Hacker News front page, researches the top 3 stories in parallel, and produces an executive summary

This creates `squadron.hcl` and a `.squadron/` directory with the encrypted vault.

## 3. Run the Starter Mission

```bash
squadron mission hn_research
```

You'll see the mission execute live in your terminal — fetching the page, processing each story in parallel, then summarizing.

## 4. Or Launch the Command Center

```bash
squadron engage --cc
```

Launches a web UI in your browser where you can run missions, edit variables, watch live execution, and browse run history. Squadron runs in the background and starts automatically on boot.

Stop it with:

```bash
squadron disengage
```
